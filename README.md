# TO INSTALL

Clone Repository, and run.

`shards update`
`./build.sh`

Move the binaries to your path. ex.

`sudo mv adlink /usr/local/bin/`
`sudo mv auauth /usr/local/bin/`


# Oneliner.

`shards update && ./build.sh && sudo mv adlink /usr/local/bin/ && sudo mv auauth /usr/local/bin/`
